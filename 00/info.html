Boards Importer 1.0.1
(Август 2025)

Плагин Boards Importer для Eagle позволяет быстро импортировать все пины из указанных досок Pinterest в вашу библиотеку Eagle. Он автоматически скачивает медиа (изображения и видео), сохраняет название, оригинальную ссылку, теги и примечания. При повторном импорте плагин определяет и пропускает уже импортированные пины, обновляя теги. Плагин также обнаруживает медиа, повторяющиеся внутри одной доски.

Почему плагин лучше чем расширение для Chrome от Eagle

  • Chrome-расширение пропускает story-pins (Idea Pins), тогда как плагин импортирует их вместе с обычными пинами.
  • Расширение работает с одной доской за раз, а плагин позволяет импортировать сразу несколько досок.
  • Chrome-расширение сохраняет только заголовок пина; плагин сохраняет также описание и ссылку на источник.
  • Расширение добавляет тег только одной доски, а плагин помечает все доски, где встречается этот пин.
  • Плагин обнаруживает и показывает медиа, которые повторяются внутри одной доски, тогда как расширение не выявляет такие дубликаты.

Интерфейс плагина

В правом верхнем углу плагина находится переключатель языка (EN/RU) и иконка ⓘ для вызова этой справки (открывается в системном браузере).

Требования

  • ✅ Windows (полная функциональность)
  • ⚠️ macOS (ограниченный режим)
  • Eagle версии 4.0.0 или выше.
  • Интернет для взаимодействия с сервисами Pinterest.

Поддержка платформ

НОВОЕ в v1.0.1: Добавлена поддержка macOS! Плагин теперь работает на Windows и macOS с автоматическим определением платформы.

Пользователи macOS, обратите внимание: Из-за ограничений безопасности системы плагин работает в ограниченном режиме на macOS:

  • Импорт ограничен последними 10 досками профиля
  • Максимум 50 пинов на доску
  • Некоторые пины могут быть пропущены, если медиа не удается извлечь через API

Шаги использования

1. Авторизация

Нажмите войдите и выполните вход в Pinterest в появившемся окне браузера (плагин не хранит логин и пароль — они сохраняются в Eagle).

2. Загрузка досок

Вставьте URL профиля в поле URL профиля. Плагин запоминает последний адрес.
Нажмите Загрузить и дождитесь списка досок. При ≥10 досках (или если начальное сканирование не дало результатов) плагин попытается открыть окно для более глубокого сканирования метаданных.

3. Выбор досок

  • Отметьте нужные доски.
  • При >50 пинов включите Полный для полного импорта; если опция отключена — будут импортированы только последние ~50 пинов с каждой доски.
  • Покажется «Выбрано: N досок(и), M пинов всего».

4. Импорт

Нажмите Импорт. Плагин может открывать окна для прокрутки досок и отдельных пинов. Прогресс отображается так (пример):

Обработано: 120/200, добавлено: 100, пропущено дубликатов: 15, изменено: 5, найдено копий: 2

Пины, медиа которых уже есть в Eagle, не дублируются: теги добавляются к существующему элементу, и он попадает в «изменено». Медиа, повторяющиеся на одной и той же доске, регистрируются в «найдено копий». После окончания отображается сообщение (пример):

Готово: добавлено 180 из 200, пропущено дубликатов: 15, изменено: 5, найдено копий: 2

и воспроизводится системный звук.

Модальные окна результатов

  • Измененные элементы: При обновлении тегов появится ссылка изменено: N. Кликните, чтобы открыть окно с галереей миниатюр. Клик по миниатюре откроет элемент в Eagle.
  • Найденные копии: Ссылка найдено копий: N откроет окно со списком URL пинов, чьи медиафайлы повторяются на одной доске. Ссылки открываются в системном браузере.

Советы и устранение неполадок

  • Нет досок: проверьте, что профиль публичный и URL корректен. Некоторые профили могут требовать глубокого сканирования для обнаружения досок.
  • Если после первого нажатия Load и открытия окна со страницей профиля для глубокого сканирования доски не нашлись, попробуйте нажать Load еще раз. Иногда страница профиля в фоновом окне не успевает полностью загрузиться с первого раза, или список досок появляется после дополнительной прокрутки, которую инициирует повторное нажатие.

Boards Importer 1.0.1
(August 2025)

The Boards Importer plugin for Eagle lets you import all pins from selected Pinterest boards into your Eagle library. It downloads media (images and videos), saves the title, original link, tags, and notes. Re-importing only updates tags for existing items. The plugin also detects media that appears multiple times within the same board.

Why the plugin is better than the Eagle Chrome Extension

  • The Chrome extension skips story-pins (Idea Pins), while the plugin imports them along with regular pins.
  • The extension imports one board at a time; this plugin handles multiple boards at once.
  • The extension saves only the pin title; this plugin also saves the description and source link.
  • The extension tags pins with a single board name; this plugin tags them with all boards where they appear (if re-importing the same pin under a different board).
  • The plugin detects and reports media that appears multiple times within the same board, while the extension doesn't identify such duplicates.

Plugin Interface

In the top-right corner of the plugin, you'll find a language switcher (EN/RU) and an ⓘ icon to open this help page (which opens in your system browser).

Requirements

  • ✅ Windows (full functionality)
  • ⚠️ macOS (limited mode)
  • Eagle version 4.0.0 or higher.
  • Internet access for interacting with Pinterest services.

Platform Support

NEW in v1.0.1: macOS support added! The plugin now works on both Windows and macOS with automatic platform detection.

macOS users please note: Due to system security restrictions, the plugin operates in limited mode on macOS:

  • Import limited to last 10 boards per profile
  • Maximum 50 pins per board
  • Some pins may be skipped if media cannot be extracted via API

Usage Steps

1. Authorization

Click log in and sign in to Pinterest in the window that appears (login and password are stored in Eagle, not in the plugin).

2. Load Boards

Paste your profile URL into the Profile URL field. The plugin remembers it.
Click Load and wait. For 10+ boards (or if the initial scan yields no results), the plugin will attempt to open a window for deeper metadata fetching.

3. Select Boards

  • Check the boards you want.
  • For >50 pins, enable Full for a complete import; if disabled, only the last ~50 pins per board will be imported.
  • The counter shows “Selected: N board(s), M pins total”.

4. Import

Click Import. The plugin may open windows for scrolling boards and individual pin metadata. Progress (example):

Processed: 120/200, added: 100, skipped duplicates: 15, modified: 5, copies found: 2

Pins whose media already exists in Eagle are not duplicated: tags are added to the existing item, and it's counted under “modified”. Media appearing multiple times on the same board are logged under "copies found". Upon completion, a message is displayed (example):

Done: added 180 of 200, skipped duplicates: 15, modified: 5, copies found: 2

and a system sound plays.

Result Modals

  • Modified Items: When tags update, a modified: N link appears. Click to open a modal gallery with thumbnails. Clicking a thumbnail opens the item in Eagle.
  • Copies Found: The copies found: N link opens a modal listing pin URLs whose media files are repeated on the same board. Links open in your system browser.

Tips & Troubleshooting

  • No boards found: ensure your profile is public and the URL is correct. Some profiles might require the deeper scan to list boards.
  • If, after the first click on Load and the opening of the profile page window for deep scanning, no boards are found, try clicking Load again. Sometimes the profile page in the background window may not fully load on the first attempt, or the board list might appear after additional scrolling initiated by a second click.
