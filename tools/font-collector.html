Font Collector 1.0

Utility for automatically collecting fonts in After Effects, Photoshop and Illustrator.

No more “Missing Fonts” when opening older projects. The script scans text layers, finds the required font files on the system (Windows/macOS) and copies them into a folder next to the project. Ideal for archiving.

Download script

Features

Saves fonts from your project

Run the script and it will collect every font used in the project text so you can reopen the archive later with no missing font warnings.

  • Multi-app. One .jsxbin works in all three apps and auto-detects the host.
  • Deep search. Scans not only system folders but also hidden Adobe and CoreSync directories.
  • Smart family fetch. Tries to grab the whole family so you don’t lose Bold, Italic, etc.
  • Full report. Creates fonts_report.txt with found files and missing-font warnings.
