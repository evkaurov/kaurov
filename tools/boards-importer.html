Boards Importer 1.0.1

Import Pinterest boards into your Eagle library with media, metadata, and duplicate protection.

Boards Importer pulls images, videos, and story pins (Idea Pins) from multiple Pinterest boards at once. It preserves titles, descriptions, source links, and tags while skipping existing items by updating their tags. Built to outperform the Eagle Chrome extension with smarter duplicate detection.

Install plugin Visit plugin page

Features

Comprehensive Pinterest importing for Eagle

Boards Importer is the most complete Pinterest import solution for Eagle, bringing in every pin with metadata and safeguards that the official Chrome extension misses.

  • Multi-board import. Select and import multiple Pinterest boards simultaneously.
  • Complete media support. Downloads images, videos, and story pins (Idea Pins).
  • Rich metadata. Saves titles, descriptions, source links, and tags for every pin.
  • Smart duplicate handling. Detects existing items, updates tags, and reports copies within the same board.
  • Full vs. quick mode. Choose between full board import or only the latest ~50 pins.
  • Bilingual interface. Interface available in English and Russian.

Why this beats the Eagle Chrome extension:

  • Story-pin support. Imports Idea Pins that the extension skips.
  • One-and-done. Handles multiple boards in one operation instead of one-by-one.
  • Complete context. Preserves descriptions and source links, not just titles.
  • Better tagging. Tags pins with every board where they appear.
  • Duplicate awareness. Reports media repeated within the same board.
