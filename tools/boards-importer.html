Boards Importer 1.0.1

Moves Pinterest boards into Eagle — neatly and hands-free.

Boards Importer helps keep your Pinterest references inside Eagle. The plugin downloads images and videos, brings over descriptions, source links, and tags so you keep context and can find assets later.

Great for collecting references for design, motion, branding, and presentations.

Install plugin Visit plugin page

Features

  • Imports entire Pinterest boards. Select one or multiple boards — the plugin grabs everything for you.
  • Handles all content types. Supports images, videos, and Idea Pins.
  • Keeps useful info. Title, description, source link, and tags all stay in Eagle.
  • Avoids duplicates. If a pin was imported before, the plugin finds it and updates tags.
  • Partial imports. Pull only the latest ~50 pins when you just need a refresh.
  • Interface in English and Russian.

How it differs from the Eagle Chrome extension

  • Imports Idea Pins. Captures the pins the official extension ignores.
  • Multiple boards at once. Pulls several boards in a single run.
  • Saves descriptions and links. Not just images.
  • Tag handling that stays correct. Even when a pin appears on multiple boards.
  • Shows duplicates. Surfaces repeats within the same board.
