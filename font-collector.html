What it does

Keep every font under control

Font Collector helps you avoid surprises before final render or project handoff. Run it once and receive a compact text report with every typeface currently used in the project.

  • Scans the whole project and collects unique font families from text layers.
  • Shows how many times each font appears and which comps it belongs to.
  • Exports a UTF-8 report that you can share with clients or archive.
  • Highlights potentially missing fonts so you can swap or relink them early.
Quick start
  1. Open your project and run Font Collector 1.0.
  2. Choose where to save the TXT report.
  3. Send the exported list to clients or teammates.
